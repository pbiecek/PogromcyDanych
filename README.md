Installing PogromcyDanych package:
To get started, install the latest version of **PogromcyDanych**:
```{Ruby}
if (!require(devtools)) {
    install.packages("devtools")
    require(devtools)
}
install_github("pbiecek/PogromcyDanych")
```

or use CRAN version:

```{Ruby}
install.packages("PogromcyDanych")
```
